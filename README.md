This project is based on Image Classification using Convolutional Neural Networks (CNN) with TensorFlow and Keras. The model was trained using the CIFAR-10 dataset, which contains different categories of images such as airplanes, cars, cats, dogs, ships, and trucks. The project involves data preprocessing, building the CNN architecture, training the model, testing its performance, and evaluating the accuracy. Convolution and pooling layers were used to extract important image features, while dense layers were used for classification. The project demonstrates the practical implementation of Deep Learning techniques for image recognition and classification tasks using Python and Google Colab.

TOOLS:Python, TensorFlow, Keras, Matplotlib, Google Colab, CIFAR-10 Dataset, NumPy.

The project was implemented using the [Google Colab] platform, which provides an online environment for writing and executing Python code. Google Colab was used to build, train, and evaluate the Convolutional Neural Network (CNN) model efficiently without requiring local system setup. It also helped in visualizing outputs, saving the notebook, and managing the deep learning workflow easily.
